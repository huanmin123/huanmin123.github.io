普通上传文件方式

@RequestMapping(value = "/upload",method = RequestMethod.POST)
    @ResponseBody
    public void  parameter(HttpServletRequest request) throws Exception {
        System.out.println("文件上传");
        //使用fileupload组件完成文件上传
        //上传的位置
        String path= request.getSession().getServletContext().getRealPath("/uploads/");
        //判断  路径是否存在
        File file =new File(path);
        if (!file.exists()){
            //创建该文件夹
            file.mkdirs();
        }

        //解析request对象 ,获取上传文件项
        DiskFileItemFactory factory =new DiskFileItemFactory();
        ServletFileUpload upload=new ServletFileUpload(factory);
        //解析request
        List<FileItem> fileItems = upload.parseRequest(request);
        //遍历
        for (FileItem item:fileItems) {
            //进行判断,当前item对象是否是上传文件项
            if(item.isFormField()){
                //说明普通表单项

            }else{
                //说明上传文件项
                //获取上传文件的名字
                String filename=item.getName();
                //把文件的名称设置为唯一值 ,防止以后文件被覆盖
                String uuid = UUID.randomUUID().toString().replace("-","");
                filename=uuid+"_"+filename;  //将文件名进行修改变成唯一
                //完成文件上传
                item.write(new File(path,filename));
                //删除临时文件
                item.delete();
            }
        }

    }

SpringMVC上传文件方式

    @RequestMapping(value = "/upload", method = RequestMethod.POST)
    @ResponseBody
    public void parameter(MultipartFile uploadFile, HttpServletRequest request) throws IOException {
        //使用fileupload组件完成文件上传
        //上传的位置   这句代码就是将文件上传到当前服务器的根目录下uploads文件夹里添加
        // 当然也可以以指定本地位置   比如 String path1="D:\\uploads\\";
        String path = request.getSession().getServletContext().getRealPath("/uploads/");

        //判断  路径是否存在  如果没有此文件那么就创建一个uploads
        File file = new File(path);
        if (!file.exists()) {
            //创建该文件夹
            file.mkdirs();
        }
        //判断文件上传表单是否是空
        boolean empty = uploadFile.isEmpty();
        //如果是空 的file表单那么 跳过
        if (!empty) {
            //获取上传文件的名字
            String filename = uploadFile.getOriginalFilename();
            //随机生成32位文件名称  防止以后上传的文件被覆盖
            String uuid = UUID.randomUUID().toString().replace("-", "");
            filename = uuid + "_" + filename;  //随机生成32位文件名称 加上原来文件名  将文件名变成唯一
            //完成文件上传
            uploadFile.transferTo(new File(path, filename));
        }

    }

SpringMVC跨服务器上传文件方式

分服务器的目的:

在实际开发中,我们会有很多处理不同功能的服务器,例如

应用服务器:负责部署我们的应用

数据库服务器,运行我们的数据库

缓存和消息服务器,负责处理大量并发访问的缓存和消息

文件(图片,视频 …)服务器:负责存储用户上传文件的服务器

而我们主要的目的是在应用服务器 将文件上传到 其他服务器里

  1. 第一步修改要上传文件的目标服务器的Tomcat 里的conf 目录下的web.xml配置文件 在这个文件里找到到 以下代码 让进行补充 大概在110~140行左右位置

        <servlet>
        <servlet-name>default</servlet-name>
        <servlet-class>org.apache.catalina.servlets.DefaultServlet</servlet-class>
        <init-param>
            <param-name>debug</param-name>
            <param-value>0</param-value>
        </init-param>

        <init-param>
            <param-name>listings</param-name>
            <param-value>false</param-value>
        </init-param>

<!--  将以下这段代码补充进去   意思就是 关闭只读    -->
        <init-param>
        <param-name>readonly</param-name>
        <param-value>false</param-value>
        </init-param>
    <!--  -------------   -->    
        <load-on-startup>1</load-on-startup>
    </servlet>

  1. 第二步修改目标服务器的Tomcat 里的conf 目录下server.xml配置文件

    在文件最后面<Host> 标签最后 添加

    <Context path="/uploads" docBase="E:\apache-tomcat-8.5.51\webapps\uploads" reloadable="true"/>

    path: 就是你访问路径

    docBase:就是你要将上传的文件放在那里

  2. 第三步在webapps目录下创建一个uploads目录

配置完这些后这个服务器的webapps目录下uploads目录 就运行其他服务器写入文件了

以上配置是能上传到服务器里 但是如果我想将文件上传到 目标服务器的 项目里怎么办

很简单

  1. 第一步同上

  2. 第二步 将目标服务器项目 在IDEA 中配置可访问目录

    1. IDEA 在生成的target里找到对应的项目下面创建一个uploads目录

    1. 然后 项目Tomcat配置里 进行配置指定的可访问文件夹

之后重新编译下项目 将新的war包 重新放入文件服务器里 然后其他的服务器就可以将文件上传到 文件服务器里了

    @RequestMapping(value = "/upload",method = RequestMethod.POST)
    @ResponseBody
    public  void suss( MultipartFile uploadFile) throws Exception {
        //判断文件上传表单是否是空
        boolean empty =  uploadFile.isEmpty();
        //如果是空 的file表单那么 跳过
        if (!empty){
            //上传文件目标  服务器路径
            String path="http://localhost:8081/uploads/";
            // 获取上传文件的名称
            String filename = uploadFile.getOriginalFilename();
            // 把文件的名称设置唯一值，uuid
            String uuid = UUID.randomUUID().toString().replace("-", "");
            filename = uuid+"_"+filename;
            // 创建客户端的对象
            Client client = Client.create();
            // 和图片服务器进行连接
            WebResource webResource = client.resource(path + filename);
            // 上传文件
            webResource.put(uploadFile.getBytes());

        }



    }

后台MVC代码

 @RequestMapping(value = "/upload",method = RequestMethod.POST)
    @ResponseBody
    public void  parameter(List<MultipartFile> uploadFile, HttpServletRequest request) throws IOException {
        //MultipartFile 自动获取 上传文件类型的表单
        //上传的位置   这句代码就是将文件上传到当前服务器的根目录下uploads文件夹里添加
        // 当然也可以以指定本地位置   比如 String path1="D:\\uploads\\";
        String path= request.getSession().getServletContext().getRealPath("/uploads/");
        //判断  路径是否存在  如果没有此文件那么就创建一个uploads
        File file =new File(path);
        if (!file.exists()){
            //创建该文件夹
            file.mkdirs();
        }
        for (MultipartFile multipartFile : uploadFile) {
            //判断文件上传表单是否是空
            boolean empty = multipartFile.isEmpty();
            //如果是空 的file表单那么 跳过
                if (empty){
                    continue;
                }
                //获取上传文件的大小  单位字节
            long size = multipartFile.getSize();
                // 上传文件不能大于50mb   跳过
                if (size>=10485760*5){
                    continue;
                }
            //获取上传文件的名字
            String filename= multipartFile.getOriginalFilename();
            //随机生成32位文件名称  防止以后上传的文件被覆盖
            String uuid = UUID.randomUUID().toString().replace("-","");
            filename=uuid+"_"+filename;  //随机生成32位文件名称 加上原来文件名  将文件名变成唯一
            //完成文件上传
            multipartFile.transferTo(new File(path,filename));
        }

    }

        function getPath(index) {
            var url = null;
            var fileObj = document.getElementsByName("uploadFile")[index].files[0];
            if (window.createObjcectURL != undefined) {
                url = window.createOjcectURL(fileObj);

                url = window.webkitURL.createObjectURL(fileObj);
