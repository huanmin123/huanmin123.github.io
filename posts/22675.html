特备注意的地方 : 
    当使用js 的时候 注意
var on=new Option();
on.value=this.name;
on.text=this.name;
$("#shi").append(on);
这样 创建对象的方法添加 别使用text() 文本添加 否则有 时候会发现 添加不上去的情况
还有注意 请空 列表的使用
$("#shi").empty();
而且还要注意 当你使用 三级联动的使用  不要在click事件中使用empty();或者清空  而是在 change 事件中使用  否则就会发现 无法添加内容成功 一直为空  
因为 会执行两次click 和异常 change
顺序是1.click  2.change   3.click      也就是你添加值后又给清空了
如果看不懂的话这有代码  你可以很清除的发现 他们的先后顺序
<select id="tt">
    <option>1</option>
    <option>2</option>
</select>

var cli=1;
$("#tt").click(function () {
alert("触发了click 第"+cli+"次");
cli++;
})
$("#tt").change(function () {
    alert("触发了change")
})

自定义文件上传按钮

需要用到的事件 click 和 change 这两个事件
click 用于打开表单的选择文件弹窗 而 change用于选择文件后的状态

在常规的获取文件路径是获取不到本地文件的绝对路径的会被游览器隐藏的

只能通过以下方法获得本地的文件的地址 而不是路径 虽然你不认识但是 src认识

实现步骤

将file上传隐藏

  1. 创建一个图片框当点击这个图片框的时候触发click事件

然后使用$(“file”).click() 打开file上传的按钮进行选文件

  1. 当选择完后通过$(“ file “).change(function () { … } 来改变图片框内的图片
//获取本地图片 路径的办法  这个路径 不是实际的路径而是url=http://localhost:8080/文件地址 可以被img标签解析
function getPath() {

  var url = null;
//btn_1 是file 
  var fileObj = document.getElementById("btn_1").files[0];
  if (window.createObjcectURL != undefined) {
    url = window.createOjcectURL(fileObj);
  } else if (window.URL != undefined) {
    url = window.URL.createObjectURL(fileObj);
  } else if (window.webkitURL != undefined) {
    url = window.webkitURL.createObjectURL(fileObj);

  }
  return url
}



$("#img_1").attr("src",getPath())
