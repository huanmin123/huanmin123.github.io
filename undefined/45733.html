java环境搭建

JVM

JVM: 是JAVA虚拟机，他的作用是将java字节码解释为各个操作系统平台能够执行的指令，

是java最具吸引力的特性之一,我们编写代码,都运行在JVM中

跨平台:任何软件,都必须要运行在操作系统之上,而我们使用java编写的软件可以运行在任何操作系统上,这个特性称为java语言的跨平台特性,该特效是由JVM实现的,我们编写的程序运行在JVM中,而JVM运行在操作系统上.

JRE和JDK

  • JRE :是java程序的运行环境,包含JVM和运行时所需要的核心类库(比如:java.lang,java.util、java.IO。)

  • JDK : 是java开发工具包 其中包含JRE和JVM 和一些java工具

    (比如javac.exe编译器,jvisualvm.exe性能分析工具)

下载完后解压 将bin路径添加到环境变量中

打开资源管理器–>右击“计算机”–>“属性”–>“高级系统设置”–>“高级”标签–>“环境变量”按钮–>“系统变量”–>“新建”按钮

变量名：JAVA_HOME

变量值：C:\Program Files\Java\jdk1.8.0_73

“新建”按钮

变量名：CLASSPATH

变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

系统变量中选择Path，单击“编辑”。在原有变量值的最后，添加

变量名：Path

变量值：%JAVA_HOME%\bin;

注意分号 path 中如果有分号就不用添加了 如果没有就添加 然后打开cmd 输入javac 如果显示内容那么就成功了

编写你的第一个程序Hell World

java开发分为三个步骤

  1. 编写

    java源码

  2. 编译

    生成.class文件

  3. 运行

    运行.class文件

在你的桌面上创建一个文本文件 文件名是HelloWorld.java

然后打开记事本 编写代码 注意:class 的名称要和文件的名称一致

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("你好世界");
    }
}

之后打开命令行输入javac 文件名 将文件编译成.class文件

这里注意的是 你要先使用命令 进入文件目录才能编译否则找不到文件

然后在当前目录下生成了一个HelloWorld.class 文件

之后在命令行中 使用 java HelloWorld 就可以生成结果了

注意 这里 java不带c 文件不带后缀.class

注意这里 为什么会乱码 因为是中文 所以在最开始编译的时候 需要指定 编码格式

javac -encoding utf-8 HelloWorld.java
