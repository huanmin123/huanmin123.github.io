CSS-选择器大全

基本选择器

id选择器

  标签的id属性唯一不能重复 选择指定的id标签

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        #div_id{
           width: 100px;
            height: 100px;
            border: 1px salmon solid;
        }
    </style>
</head>
<body>
<div id="div_id"></div>
</body>
</html>

class选择器

  标签的class可以重复 选择所有指定的class的标签

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
       .div_class{
            width: 100px;
            height: 100px;
            border: 1px salmon solid;
        }
    </style>
</head>
<body>
<div class="div_class"></div>
<div class="div_class"></div>
</body>
</html>

标签选择器

  选择所有指定的标签

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
       div{
            width: 100px;
            height: 100px;
            border: 1px salmon solid;
        }
    </style>

选择器分组

假设有两个标签 都要需要 字体加粗 但是如果一个一个去设置太麻烦了

这时候就可以使用 选择器分组 让多个标签共享同一个样式

语法 a,b,c{ } 使用逗号隔开

比如


<h1 class="h1_class" id="h1_id">abc</h1>
<p class="p_class" id="p_id">abc</p>

        h1,p{
            color: salmon;
        }

或者

    .h1_class,#p_id{
            color: salmon;
        }

在或者

   .h1_class,#p_id:nth-of-type(1){
            color: salmon;
        }

任意选择器都行

层次选择器

后代选择器语法

后代选择器语法: E F{ }

指定E标签内的相关的所有F元素

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        #div_id{
            width: 100px;
            height: 100px;
            border: 1px #fa6a60 solid;
        }
        #div_id p{
            font-size: 20px;
            color: chartreuse;
        }
    </style>
</head>
<body>
<div  id="div_id">
    <p>abc</p>
    <div>
        <p>abc</p>
    </div>
</div>
</body>
</html>

子选择器

子选择器语法: E>F{ }

就是E标签内的第一层F元素

<!DOCTYPE html>

<body>
<div  id="div_id">
    <p>abc</p>
</div>
</body>
</html>

相邻兄弟选择器

相邻兄弟选择器语法: E+F{ }

属性选择器

E 代表标签 attr代表属性值

查找指定属性对应的标签E[attr]

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        input[name]{
            font-size: 20px;
            color: #ffafb9;
        }
    </style>

<body>

    <input type="text" name="name" value="huanmin"><br/>
    <input type="text" name="age" value="22">
</form>
</body>
</html>

结构伪类选择器

利用后代选择器 加上结构伪类 就是操作元素内的元素

如果不使用后代选择器的话那么就是在同一层级选择 E:xxx{}

还有一个选择器E:not(?){ }

            width: 100px;
            height: 30px;
           color: sandybrown;
        }

各种选择器组合的方法参考

鼠标状态 配合 后代选择器 和结构伪类 选择器进行组合

   <style>
#a_id:hover #div_id:nth-of-type(1){
    ......
}
   </style>

<a id="a_id"><div id="div_id"> </div></a>

这个意思 当鼠标悬浮 在a标签的时候 div就会发生变化

#a_id:hover+#div_id{
 ............   
}
<a href="#" id="a_id">来鼠标放过来</a>
<div id="div_id"> </div>

这个意思 当鼠标悬浮a的时候 他下面的div变动
